# MSc Data Science & Analytics — USP Coursework

R exercises and analyses from my MSc in Data Science & Analytics at
Universidade de São Paulo (2024–present).

## What's here

Organized by course module. Each folder contains the exercise prompt,
data (when shareable), my solution notebook, and notes on approach.

## Why public

Two reasons. First, a public repo keeps me honest — code I'd be
embarrassed to show doesn't make it in. Second, other working
professionals returning to graduate study may find the trade-offs useful
(how to balance depth vs. time, what tools carry over from industry,
where academic rigor actually sharpens applied practice).

## Structure

- `module-01-<name>/` — exercises and solutions
- `module-02-<name>/` — ...
- `utils/` — helper functions reused across modules

## Not a tutorial

This is coursework, not a lecture. Solutions reflect my approach, not
necessarily the "right" one. If you spot a better method, issues and
PRs are welcome.
